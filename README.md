# Epitech_my_sokoban

The goal of this project is to recreate a sokoban-type game in C using the CSFML lib. 

    USAGE

      ./navy [MAP_PATH]

    DESCRIPTION

      [MAP_PATH] : Path to the map
      
    COMMANDS
    
      [ARROW KEYS] : Make the player move
      
      [R KEY] Reset the game
